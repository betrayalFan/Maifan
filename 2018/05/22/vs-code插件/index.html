vs code插件

发表于 2018-05-22 | 分类于 编程工具

vs code 代码提示工具

1、jQuery Code Snippets
jQuery代码提示工具

2、Path Intellisense/Path Autocomplete
路劲默认补全

3、Auto Rename Tag
修改html标签，自动帮你完成尾部闭合标签的同步修改

4、VueHelper
Vscode中最好的VUE代码片段插件，不仅包括VUE2所有API，还包含VUE路由器2和VUEX 2代码

5、JavaScript Snippet Pack/JavaScript (ES6) snippets
JavaScript代码提示(后者包括ES6语法)

6、HTML CSS Support

7、markdownlint
markdown 语法提示器

vs code 编程助手

1、View In Browser
浏览器视图。扩展cs code以查看浏览器中的HTML文件
查看方式：打开命令列表（按F1或CTRL+Shift +P）

2、Live Server
浏览器和代码同步

3、Live Sass Compiler/Easy Sass
sass编译

4、Git History
可以查看Git log、file、和line历史记录

5、Markdown PDF
将markdown文档转换成pdf的利器

6、filesize
会在左下角显示文件大小

vs code 个性化插件

主题

深色主题

1、Atom One Dark
Atom的默认主题，熟悉Atom的可以继续熟悉了！

2、vardandi Alter
黑底白字，简约者的最爱

3、Monokai
彩色主题，色彩鲜艳但是看着很舒服

4、Monokai Dimmed
彩色主题，和Monokai同系列，但是色彩感淡一点

浅色主题

1、vardandi
白底黑字，简约者的最爱

文件图标

1、VSCode Icons

2、Material Icon Theme

3、vardandi
简约图标主题，但是很明了

4、VSCode Great Icons
对展开的文件有变色处理，很容易分辨

代码样式

1、beautiful
格式化代码的工具，是代码格式更好看
2、JavaScript Atom Grammar
使用atom风格的语法高亮、对于习惯浏览atom风格代码高亮的人实用。
