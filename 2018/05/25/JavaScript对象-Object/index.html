一、JavaScript对象的简介

1、js中5种基本数据类型

  • String 字符串
  • Number 数值
  • Boolean 布尔值
  • Null 空值
  • Undefined 未定义

2、js中除了5种基本数据类型之外的另一种数据类型——对象（Object）

  • 属于复合的数据类型，可以保存多个不同数据类型的属性
  • 只要不属于基本数据类型，那就是对象
  • 对象（Object）实例不具备多少功能，它的主要功能是存储和传输数据！

3、对象（Object）的分类

  • 内建对象——由ES标准中定义的对象，在任何的ES中的实现中都可以使用。如Math String Number
  • 宿主对象——由JS的运行环境提供的对象，目前来讲主要指浏览器提供和定义的对象。如BOM DOM
  • 自定义对象——由开发人员自己创建的对象

4、Object类型所具有的任何属性和方法也同样存在于更具体的对象中
