HTML

1.HTML

超文本标记语言（HyperText Markup Language）
“超文本”解释：页面内可以包含图片、链接，甚至音乐、程序等非文字元素
“标记语言”解释：是一套标记标签，而非编程语言

HTML基础：标题、段落、链接、图像

2.HTML5
HTML5 是为了让 HTML 更加的语义化并在移动设备上支持多媒体而设计。

HTML5新增特性
（1）、用于绘画的 canvas 元素（canvas插件）
（2）、用于媒介回放的 video 和 audio 元素
（3）、对本地离线存储的更好的支持（本地数据存储、访问本地文件、缓存引用）
（4）、新的特殊内容元素，比如 article、footer、header、nav、section
（5）、新的表单控件，比如 calendar、date、time、email、url、search

Web语义化

1.什么是语义化
书面语：用合理HTML标记以及其特有的属性去格式化文档内容
口语：可以想象为人和机器的对话，人的语言越接近机器语言，机器的反馈会越智能

2.为什么要用语义化
（1）、在没有CSS代码时，也可以呈现很好的内容结构、代码结构
（2）、利于SEO，语义化可以和搜索引擎建立良好的联系，有利于爬虫抓取更多的有效信息（爬虫利用标签来确定上下文和各个关键字的权重）
（3）、提高用户体验，如：显示页面信息的title或者显示图片信息的alt
（4）、支持多终端设备的浏览器渲染
（5）、方便团队的开发，上面提到人和机器的交流，当然也包括人和人的交流

3.写页面需要注意的地方
（1）、尽可能的多使用语义化的标签，少使用没有语义的标签。如div、span等
（2）、对语义的要求不明显的时候，如果在div和p标签之间做选择，优先选择p标签，因为p标签可以兼容特殊终端
（3）、强调文本标题之类的，用标题标签或者em（强调）、strong（重点强调）标签，不要使用CSS加粗来达到目的，当然，纯样式标签除外
（4）、使用table时，严格的使用table的结构
（5）、注意代码结构的整洁（代码格式）规范（双标签和单标签的使用）以及注释

课程问答

问 ：什么是Web，什么是HTML，CSS，JavaScript(简述)

Web

Web简单的来说就是网页的意思。
表现为三种形式，即超文本(hypertext)、超媒体(hypermedia)、超文本传输协议(HTTP)

超文本（hypertext）　　
一种全局性的信息结构，它将文档中的不同部分通过关键字建立链接，使信息得以用交互方式搜索。它是超级文本的简称。

超媒体（hypermedia）
超文本（hypertext）和多媒体在信息浏览环境下的结合。它是超级媒体的简称。用户不仅能从一个文本跳到另一个文本，而且可以激活一段声音，显示一个图形，甚至可以播放一段动画。

超文本传输协议（HTTP）
超文本在互联网上的传输协议。

HTML

HTML（HyperText Markup Language）超文本标记语言。HTML被称为超文本，是因为HTML包含有“超级链接点”，用于获取新的页面。由此可见，网页的本质就是HTML，通过结合使用其他的Web技术（如：脚本语言、CGI、组件等），可以创造出功能强大的网页。因而，HTML是Web编程的基础，也就是说万维网是建立在超文本基础之上的。

CSS

CSS层叠样式表(Cascading Style Sheets)用来表现HTML文件样式的计算机语言

javaScript

JavaScript是在用户的浏览器上运行，不需要服务器的支持而可以独立运行
JavaScript一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。
它的解释器被称为JavaScript引擎，为浏览器的一部分，广泛用于客户端的脚本语言，最早是在HTML（标准通用标记语言下的一个应用）网页上使用，用来给HTML网页增加动态功能。
